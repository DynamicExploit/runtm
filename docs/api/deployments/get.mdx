---
title: "Get Deployment"
api: "GET /v0/deployments/{deployment_id}"
description: "Get deployment status and details"
---

Retrieve the current status and details of a specific deployment.

## Path Parameters

<ParamField path="deployment_id" type="string" required>
  The deployment ID (e.g., `dep_abc123xyz`)
</ParamField>

## Response

<ResponseField name="deployment_id" type="string">
  Unique deployment identifier
</ResponseField>

<ResponseField name="name" type="string">
  Project name from manifest
</ResponseField>

<ResponseField name="state" type="string">
  Current state: `queued`, `building`, `deploying`, `ready`, `failed`, `destroyed`
</ResponseField>

<ResponseField name="url" type="string">
  Live URL (only when `ready`)
</ResponseField>

<ResponseField name="error_message" type="string">
  Error details (only when `failed`)
</ResponseField>

<ResponseField name="template" type="string">
  Template type
</ResponseField>

<ResponseField name="runtime" type="string">
  Runtime: `python`, `node`
</ResponseField>

<ResponseField name="version" type="integer">
  Deployment version number
</ResponseField>

<ResponseField name="is_latest" type="boolean">
  Whether this is the latest version
</ResponseField>

<ResponseField name="app_name" type="string">
  Fly.io app name (for domain commands)
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp
</ResponseField>

<ResponseField name="ready_at" type="string">
  When deployment became ready
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://app.runtm.com/api/v0/deployments/dep_abc123xyz" \
  -H "Authorization: Bearer runtm_xxx"
```

```python Python
import requests

response = requests.get(
    "https://app.runtm.com/api/v0/deployments/dep_abc123xyz",
    headers={"Authorization": "Bearer runtm_xxx"}
)

data = response.json()
print(f"Status: {data['state']}")
if data["url"]:
    print(f"URL: {data['url']}")
```

```javascript JavaScript
const response = await fetch(
  "https://app.runtm.com/api/v0/deployments/dep_abc123xyz",
  {
    headers: {
      "Authorization": "Bearer runtm_xxx"
    }
  }
);

const data = await response.json();
console.log(`Status: ${data.state}`);
if (data.url) {
  console.log(`URL: ${data.url}`);
}
```
</RequestExample>
