---
title: "Remove Domain"
api: "DELETE /v0/deployments/{deployment_id}/domains/{hostname}"
description: "Remove a custom domain from a deployment"
---

Remove a custom domain from a deployment. This deletes the certificate and domain configuration.

## Path Parameters

<ParamField path="deployment_id" type="string" required>
  The deployment ID (e.g., `dep_abc123xyz`)
</ParamField>

<ParamField path="hostname" type="string" required>
  The custom domain hostname to remove (e.g., `api.example.com`)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the domain was successfully removed
</ResponseField>

<ResponseField name="message" type="string">
  Status message
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://app.runtm.com/api/v0/deployments/dep_abc123xyz/domains/api.example.com" \
  -H "Authorization: Bearer runtm_xxx"
```

```python Python
import requests

response = requests.delete(
    "https://app.runtm.com/api/v0/deployments/dep_abc123xyz/domains/api.example.com",
    headers={"Authorization": "Bearer runtm_xxx"}
)

data = response.json()
if data["success"]:
    print(f"✓ {data['message']}")
else:
    print(f"✗ {data['message']}")
```

```javascript JavaScript
const response = await fetch(
  "https://app.runtm.com/api/v0/deployments/dep_abc123xyz/domains/api.example.com",
  {
    method: "DELETE",
    headers: {
      "Authorization": "Bearer runtm_xxx"
    }
  }

const data = await response.json();
if (data.success) {

  console.log(`✗ ${data.message}`);

```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true,
  "message": "Domain api.example.com removed successfully"
}
```

```json 404
{
  "detail": {
    "error": "Deployment not found",
    "deployment_id": "dep_invalid"
  }

```
</ResponseExample>

<Note>
After removing a domain, remember to also delete the DNS record at your domain registrar.
</Note>
