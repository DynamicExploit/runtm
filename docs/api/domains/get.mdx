---
title: "Get Domain Status"
api: "GET /v0/deployments/{deployment_id}/domains/{hostname}"
description: "Check the status of a custom domain"
---

Get the current status of a custom domain, including certificate status and DNS configuration.

## Path Parameters

<ParamField path="deployment_id" type="string" required>
  The deployment ID (e.g., `dep_abc123xyz`)
</ParamField>

<ParamField path="hostname" type="string" required>
  The custom domain hostname (e.g., `api.example.com`)
</ParamField>

## Response

<ResponseField name="hostname" type="string">
  The configured hostname
</ResponseField>

<ResponseField name="configured" type="boolean">
  Whether the domain is fully configured and working
</ResponseField>

<ResponseField name="certificate_status" type="string">
  Certificate status: `pending`, `issued`, `failed`
</ResponseField>

<ResponseField name="dns_records" type="array">
  Required DNS records
</ResponseField>

<ResponseField name="error" type="string">
  Error message if configuration failed
</ResponseField>

```python Python
import requests
import time

def wait_for_certificate(deployment_id: str, hostname: str):
    """Poll until certificate is issued."""
    while True:
        response = requests.get(
            f"https://app.runtm.com/api/v0/deployments/{deployment_id}/domains/{hostname}",
            headers={"Authorization": "Bearer runtm_xxx"}
        )
        data = response.json()
        
        if data["certificate_status"] == "issued":
            print(f"✓ Certificate issued for {hostname}")
            return True
        elif data["certificate_status"] == "failed":
            print(f"✗ Certificate failed: {data.get('error')}")
            return False
        
        print(f"Certificate pending... (check DNS configuration)")
        time.sleep(30)

wait_for_certificate("dep_abc123xyz", "api.example.com")
```

```javascript JavaScript
async function waitForCertificate(deploymentId, hostname) {
  while (true) {
    const response = await fetch(
      `https://app.runtm.com/api/v0/deployments/${deploymentId}/domains/${hostname}`,
      {
        headers: {
          "Authorization": "Bearer runtm_xxx"
        }
      }
    );
    
    const data = await response.json();
    
    if (data.certificate_status === "issued") {
      console.log(`✓ Certificate issued for ${hostname}`);
      return true;
    } else if (data.certificate_status === "failed") {
      console.log(`✗ Certificate failed: ${data.error}`);
      return false;
    }
    
    console.log("Certificate pending... (check DNS configuration)");
    await new Promise(r => setTimeout(r, 30000));
  }
}

waitForCertificate("dep_abc123xyz", "api.example.com");
```
