# runtm list

List all your deployments.

```bash
runtm list
```

## Options

| Option | Short | Description |
|--------|-------|-------------|
| `--state STATE` | `-s` | Filter by state |
| `--limit N` | `-n` | Maximum results (default: 50) |

## Examples

```bash
# List all deployments
runtm list

# List only ready deployments
runtm list --state ready

# List failed deployments
runtm list --state failed

# Limit results
runtm list --limit 10
```

## State Filters

| State | Description |
|-------|-------------|
| `queued` | Waiting for worker |
| `building` | Docker build in progress |
| `deploying` | Deploying to Fly.io |
| `ready` | Live and healthy |
| `failed` | Build or deploy failed |
| `destroyed` | Deployment removed |
